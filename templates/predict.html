How it works
  1. 01
    Input

    Enter a numeric Elliptic transaction ID to look up in the dataset.

  2. 02
    Graph Context

    The GNN loads the full transaction graph — 203K nodes, 234K edges — to provide structural context.

  3. 03
    Propagation

    GraphSAGE aggregates neighbor features across 2 hops to compute the node embedding.

  4. 04
    Verdict

    A softmax classifier outputs a licit / illicit prediction with confidence score.
