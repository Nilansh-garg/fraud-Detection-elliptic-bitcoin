# Fraud Detection using Graph Neural Networks

A GNN-based fraud detection app trained on the Elliptic Bitcoin Dataset.
Detects illicit vs licit blockchain transactions using GraphSAGE.

## How to Use
Enter a Transaction ID from the Elliptic dataset and the model will classify it as **Illicit** or **Licit** with a confidence score.
